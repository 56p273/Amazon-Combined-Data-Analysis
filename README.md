# Amazon Combined Data Analysis

## 📊 Project Overview
This project is an Amazon Sales Data Analysis Dashboard created using Power BI.

The dashboard helps analyze sales performance, product categories, customer insights, and revenue trends.

- Excel
- SQL

- Product Category Analysis
- Revenue Trends

- Interactive Dashboard
